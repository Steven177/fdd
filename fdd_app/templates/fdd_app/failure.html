Failure Mode

Let's categorize the failure.

False Observation

Computer vision system generates output error.

Example

Object detector misclassifies a tennis rack as a baseball bat.

Failing to observe

Computer vision system is unable to provide an output due to a limitation of the system.

Example

Object detector fails to detect any object for a given image.

Incomplete observation

Computer vision system misses certain visual information.

Example

Object detector detects some objects correctly but misses at least one critical object.

Critical quality of bounding boxes

The output of the computer vision system bears intolerable uncertainty or quality issues.

Example

The detector's bounding boxes are imprecise.

Critical quality of confidence score

The output of the computer vision system bears intolerable uncertainty or quality issues.

Example

The uncertainty of a prediction is too high.

Violation

Computer vision system’s output stands in conflict with ethical standards, rules, or regulations.

Example: The object detector classifies a person as a toilet.

Context error

System works as intended but the behavior of the system stands in conflict with the needs, goals or preferences of its user in a given context.

Example

Object detector detects a person but the user was interested in a person's clothing.
